# simple_simulationGame_AndroidStuioWithJave

안드로이드 스튜디오의 기초 개념을 이용하여 만든 시뮬레이션 게임앱입니다.

- 사용된 개념
1. 테이블 레이아웃
2. 컨텍스트 메뉴
3. 뷰 플리퍼
4. 스크롤 뷰
5. 쌍방향 인텐드 등등
